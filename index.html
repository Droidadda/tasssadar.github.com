Notable repositories

Lorris

Lorris is Qt app designed to be used when doing something with microcontrollers, like a toolbox or something. Example: you have robot with color sensor. You need to show data from that sensor somewhere. Lorris is that somewhere.

That was the simplest example. It can do much, much more. See the slightly outdated homapage for details.

Homepage: http://tasssadar.github.com/Lorris
Repository: http://github.com/Tasssadar/Lorris

Lorris_updater

Updater app for Lorris, written in C++ and it is using WinAPI only, so that it can be as small as possible. It will just check Lorris' version, download ZIP file and extract it. Small, simple.

Repository: http://github.com/Tasssadar/Lorris_updater

bashoid

Bashoid is Audio's project and I just made several addons. It is an IRC bot, written in java and it's main advantage is addon-like structure.

Repository: http://github.com/Audio/bashoid

3piControl

Program for 3pi robot (eg. ATmega328P chip), which acts like RC car. It also shows some info on built-in display. More info in README.

Repository: http://github.com/Tasssadar/3piControl

3piLib

Library for 3pi robot compiled together mostly from their libpololu and avakar's Yunimin library. It is designed to be used as single header file, just include "3piLibPack.h" and that is all. Supports motor control, display, RS232, buzzer and more. There is a wiki available.

Repository: https://github.com/Tasssadar/3piLib

MultiROM

MultiROM is a multi-boot solution for LG Optimus One. One ROM is in the external memory, other(s) are on the SD card. The most notable feature is boot manager, which allows you to choose ROM at boot time, and to play Tetris, even when your Android system is broken. See homepage and XDA thread for more info.

XDA Thread: yep, here
Homepage: http://tasssadar.github.com/multirom/
